# Fanned Hover Image Gallery

A responsive fanned image card gallery built with vanilla HTML, CSS, and JavaScript.

The gallery keeps seven portrait cards arranged like a loose deck. Desktop users can hover or focus
cards to animate the stack, touch users can tap cards, and keyboard users can tab through the cards
or use arrow keys.

## Features

- Vanilla HTML, CSS, and JavaScript
- Seven local portrait image assets
- Fanned card stack with responsive scaling
- Hover, tap, focus, and keyboard interactions
- Accessible button controls with `aria-label` and `aria-pressed`
- SEO metadata and JSON-LD
- Reduced motion support
- Context for AI copy button
- GitHub attribution link
